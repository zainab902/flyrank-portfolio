# flyrank-portfolio

FlyRank is a capstone portfolio project organized as a modular, scalable web application with AI-assisted workflows for fast design, development, and maintenance.

## Tech Stack
- Node.js (LTS)
- Git
- VS Code / Cursor
- Modular, scalable web application architecture
- AI-assisted development workflow

## Overview
- Emphasizes clean project structure, clear routing, and robust authentication patterns.
- Uses AI to audit code, generate boilerplate, optimize schemas, and debug performance issues.
- Git commits follow Conventional Commits 1.0.0 with allowed prefixes: `feat:`, `fix:`, `docs:`, `style:`, `refactor:`, `test:`, `chore:`.
